# 🏏 Cricket Stadium Ticket Booking UI

A clean, modern, and highly interactive user interface for a **Cricket Stadium Ticket Booking System**. This project showcases the transition from a Figma UI/UX concept into a functional web component using **React** and clean **CSS**.

---

## ✨ Features
* **Interactive Seat Selection:** Users can click and select available seats dynamically.
* **Real-time Booking Summary:** Automatically updates the total number of seats selected and calculates the total price in **LKR**.
* **Reset Functionality:** A dedicated quick-reset button to clear all selections instantly.
* **Responsive & Clean UI:** Designed with strong visual hierarchy and accessible color choices.

---

## 🛠️ Tech Stack & Tools Used
* **Frontend Library:** React.js (Functional Components & State Management)
* **Styling:** Custom CSS3 (Flexbox, Grid, and Hover Effects)
* **Design/Prototyping:** Figma
